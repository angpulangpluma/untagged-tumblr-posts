Find Untagged Tumblr Posts

Step 1. Enter your Tumblr URL.

Specifically, I need your hostname. This is usually the URL of your site, although it can be a custom URL as well. Learn more.

Here's part of the Tumblr API documentation which explains this fully:

Each blog has a unique hostname. The hostname can be standard or custom.

  • Standard hostname: the blog short name + .tumblr.com.

    Example: greentype.tumblr.com

  • Custom hostname: Anything at all, as determined by a DNS CNAME entry.

    Example: www.davidslog.com

About this site

I created this site because I was dissatisfied with my existing tool for finding untagged posts on somebody's Tumblr blog. It involved running a Python script and getting your own API key, which is sufficiently many steps to keep a lot of people away. You can read more about the motivation behind this on my blog.

I’m not affiliated with Tumblr; I’m just abig fan of their stuff. I don’t work with them and they don’t endorse this site. Any bugs are mine, not theirs.

If you find any bugs, then please email me (contact details), or if you prefer, you can submit a pull request on GitHub.
